# Filter Gate Operational Protocol

When a new token pool is initialized, run these deterministic tests before invoking any cognitive reasoning engines:

1. **Verification Check:** Inspect the contract status. If `is_verified == False`, immediately abort execution and log the address to the local blacklist database tool.
2. **Liquidity Check:** Query the pool depth. If `liq_usd` is less than $2,500.00, immediately flag as "ABORTED" due to low pool depth.
3. **Short-Circuit Action:** If either condition triggers, bypass downstream nodes, write the exact failure reason to the state schema, and terminate the graph instance.
